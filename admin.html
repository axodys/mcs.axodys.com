github publishing

Enter your GitHub details once. When you click publish, posts are committed directly to your repo — no export needed.

Token needs repo scope. Create one at github.com → Settings → Developer Settings → Personal Access Tokens → Fine-grained tokens (set Contents: read & write for your repo).

image hosting (cloudinary)

Images are uploaded to Cloudinary and stored as URLs — no base64 bloat in posts.json.

Create a free account at cloudinary.com, then go to Settings → Upload → Upload presets and create an unsigned preset. Enter your cloud name and preset name below.
